weborf

Minimal HTTP server to share your files

Downloads | qweborf | Cache

Weborf is a lightweight webserver designed to rapidly share directories. Runs on GNU/Linux systems and is meant to solve this problem.

It comes with a GUI called qweborf to make it easier to know the IP address, traverse the NAT, use authentication.

Features

  • Qt GUI or and command line
  • Webdav support
  • Cache to speed up webdav operations
  • SSL support
  • (x)inetd support
  • Compress directories on the fly
  • CGI support
  • systemd integration
